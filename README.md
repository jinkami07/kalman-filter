# ロボット関節モータ — カルマンフィルタ推定

エンコーダで**角度のみ**を観測しながら、カルマンフィルタが**角度 + 角速度を同時推定**するしくみをインタラクティブに学ぶツール。

**公開サイト：** https://jinkami07.github.io/kalman-filter/

## 概要

- PD 制御による正弦波軌道追従シミュレーション（関節モータ）
- カルマンフィルタで状態 `[θ, ω]`（角度・角速度）を推定
- 角速度は直接観測できないが、運動方程式（A 行列）を通じて間接推定
- 全計算を JavaScript で実装 — サーバー不要、`index.html` 単体で動作

## 操作

| スライダー | 意味 | 大きくすると… |
|---|---|---|
| **R** | 観測ノイズ分散（センサの不確かさ） | センサを信じない → 推定が滑らか |
| **Q_θ** | 角度プロセスノイズ | センサへの追従が速くなる |
| **Q_ω** | 角速度プロセスノイズ | 角速度推定の追従が速くなる |
| **σ** | シミュレーション上のエンコーダノイズ | 観測点のばらつきが増える |

- 「🔄 ノイズ再生成」で別の乱数シードのデータを生成
- 「▶ アニメーション」でフィルタの更新過程をリアルタイム再生（ループ）

## 状態空間モデル

$$\mathbf{x}_{k+1} = A\mathbf{x}_k + Bu_k + \mathbf{w}_k, \quad z_k = H\mathbf{x}_k + v_k$$

| 行列 | 次元 | 内容 |
|---|---|---|
| A | 2×2 | 状態遷移（離散化運動方程式） |
| B | 2×1 | 制御入力（トルク） |
| H | 1×2 | 観測行列（角度のみ取り出す） |
| Q | 2×2 | プロセスノイズ共分散 |
| R | スカラー | 観測ノイズ分散 |
