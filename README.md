# カルマンフィルタ — インタラクティブ学習

ブラウザで開くだけで、パラメータをいじりながらカルマンフィルタの挙動を視覚的に学べます。

## デモ

`index.html` をブラウザで開く（サーバー不要）。

## 操作方法

| スライダー | 意味 | 動かすと… |
|---|---|---|
| **R** (観測ノイズ) | センサー精度 | 大 → フィルタが滑らか・応答遅め |
| **Q** (プロセスノイズ) | システムの不確かさ | 大 → 観測値に素早く追従 |
| **P₀** (初期推定誤差) | 初期の不確かさ | 大 → 序盤のゲインが高くなる |
| **σ_obs** (観測ノイズ標準偏差) | シミュレーションのノイズ量 | 大 → 観測点のブレが増える |

「再生成」で乱数シードを変え、「アニメ」でフィルタの更新過程を 1 ステップずつ見られます。

## 表示内容

- **青線**: 真値（正弦波）
- **赤点**: ノイズが乗った観測値
- **緑線**: カルマンフィルタの推定値
- **緑帯**: 推定誤差 ±√P（分散の平方根）
- **下グラフ**: カルマンゲイン K の推移（0 に収束していく様子）

## カルマンフィルタの更新式（1次元）

```
① 予測:
    x̂⁻ₖ = x̂ₖ₋₁
    P⁻ₖ  = Pₖ₋₁ + Q

② 更新:
    Kₖ  = P⁻ₖ / (P⁻ₖ + R)          ← カルマンゲイン
    x̂ₖ  = x̂⁻ₖ + Kₖ(zₖ − x̂⁻ₖ)     ← 観測で補正
    Pₖ  = (1 − Kₖ) P⁻ₖ             ← 分散を更新
```

K = 0 → 観測を無視、K = 1 → 観測をそのまま採用。
R が小さい（センサーが正確）ほど K が大きくなり、観測値を積極的に使う。
