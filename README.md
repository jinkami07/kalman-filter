# カルマンフィルタ — インタラクティブ学習

スライダーを動かすだけでカルマンフィルタの挙動をリアルタイムに確認できる学習ツール。
**Kalman フィルタの計算は Python（Flask + NumPy）が担当**し、ブラウザは描画のみを行う。

## セットアップ

```bash
pip install flask numpy
python server.py
# → http://localhost:5000 をブラウザで開く
```

## 操作

| スライダー | 意味 | 大きくすると… |
|---|---|---|
| **R** (観測ノイズ) | センサーの不確かさ | 観測を信じない → 推定が滑らか |
| **Q** (プロセスノイズ) | システムの不確かさ | 観測に素早く追従 |
| **P₀** (初期誤差分散) | 初期の不確かさ | 序盤のカルマンゲインが高くなる |
| **σ** (観測ノイズ標準偏差) | ノイズ量 | 赤点のブレが増える |

「🔄 再生成」で別の乱数シードのデータを生成。
「▶ アニメ」でフィルタの更新過程を 1 ステップずつ再生。

## 表示内容

| 色 | 意味 |
|---|---|
| 青線 | 真値（正弦波の重ね合わせ） |
| 赤点 | 観測値（ノイズあり） |
| 緑線 | カルマンフィルタの推定値 |
| 緑帯 | 推定誤差 ±√P |
| 下グラフ（オレンジ）| カルマンゲイン K の推移 |

## カルマンフィルタの式（1 次元）

```
① 予測
    x̂⁻ₖ = x̂ₖ₋₁
    P⁻ₖ  = Pₖ₋₁ + Q

② カルマンゲイン
    Kₖ = P⁻ₖ / (P⁻ₖ + R)

③ 更新
    x̂ₖ = x̂⁻ₖ + Kₖ (zₖ − x̂⁻ₖ)
    Pₖ  = (1 − Kₖ) P⁻ₖ
```

K → 0：観測を無視、K → 1：観測をそのまま採用。
R が小さい（センサー精度が高い）ほど K が大きくなる。
